Gitbook & Github

Gitbook & Github

创建 gitbook 后，可以使用客户端编写文档，也可以直接在web端编写，但是因为墙的存在体验非常不好。

我们可以将 gitbook 关联到 github，这样就可以在常用的编辑器里编写，然后使用 git 上传到 github 。像管理代码一样简单!

在新版本的 gitbook 界面，关联也很简单。
